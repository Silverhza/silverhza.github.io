## 推荐工作流

1. 你发我一篇论文（PDF、arXiv 链接、标题都可以）
2. 我产出一篇结构化总结
3. 你确认后，我发布到 blog
4. 如有必要，再额外提取 2~3 个知识点卡片

## 文章模板风格

这个站默认采用偏 GitHub 风格的极简排版：

- 白底 / 深色双主题
- 强调可读性与检索效率
- 适合公式、代码、论文链接和结构化总结
